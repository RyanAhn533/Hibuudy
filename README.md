# Hi-Buddy · 발달장애인을 위한 AI 하루 일정 도우미

> 코디네이터가 일정을 설계하면, 발달장애인이 하루 종일 혼자 따라할 수 있는 AI 기반 생활 지원 앱

---

## 배경 및 동기

2025년 서울청년기획봉사단 **온쉐어(溫Share)** 팀장으로 활동하며,
발달장애인을 위한 AI 요리 챗봇 **온쿡(OnCook)** 을 단독 개발하고 현장에 투입했습니다.

- 서부장애인종합복지관 자립주거팀 발달장애인 5명 대상 쿠킹클래스 2회 진행
- LG Hello Vision 기업 연계, 헬로TV 뉴스 취재
- 참여자 만족도 **4.83 / 5점**, SNS 조회수 **16,000회 이상**

이 결과보고서를 접한 **연세대학교 대학원 HEART Lab** 에서 연락이 와,
LLM + AAC(보완대체의사소통) 기술 접목 방법론을 주제로 단독 강연을 진행했습니다. (2025.12)

이후 쿠킹 챗봇에서 더 나아가, **발달장애인의 하루 전체를 지원하는 일정 관리 앱**으로 확장한 것이 Hi-Buddy입니다.

---

## 주요 기능

### 코디네이터 화면 (일정 설계)
- 자연어로 하루 일정 입력 → GPT가 구조화된 일정표로 자동 변환
- 요리 슬롯: 메뉴별 음식 사진(Google 이미지 검색 + GPT Vision 필터링) 선택
- 운동/여가 슬롯: 발달장애 친화 유튜브 영상 자동 추천 및 선택
- 안내 문장(guide_script) 수동 편집 또는 GPT 자동 생성

### 사용자 화면 (따라하기)
- 현재 시각 기준 활동 **한 가지만** 크게 표시 (집중 유도)
- 정시 자동 알람 + TTS 음성 안내 (JS 타이머 기반, Streamlit 재시작 없이 동작)
- 요리: 메뉴 선택 → 준비물/재료/단계별 레시피 표시
- 운동: 앉아서/서서 루틴 선택 후 단계별 안내
- 옷 입기, 여가, 휴식 등 8가지 활동 타입 지원

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프론트엔드 | Streamlit, streamlit-autorefresh |
| AI | OpenAI GPT-4.1-mini (일정/안내문 생성), GPT Vision (이미지 필터링), gpt-4o-mini-tts (음성 합성) |
| 이미지 검색 | Google Custom Search API + PIL 이미지 검증 |
| 영상 검색 | YouTube Data API v3 + GPT 쿼리 생성 |
| 음성 알람 | JavaScript setInterval + Base64 오디오 임베딩 |
| 데이터 | JSON 기반 일정 파일 (schedule_today.json) |

## 접근성 설계 원칙

발달장애인/노인이 혼자서도 사용할 수 있도록 다음 원칙을 적용했습니다.

- **UI**: 폰트 40px 이상, 버튼 전체 너비, 한 화면에 한 가지 정보
- **언어**: 존댓말, 문장 20~45자 이내, 단계별 한 동작씩
- **레시피**: 칼 사용 최소화(가위/손 대체), 약불/중불만 사용, 시간 단순화
- **음성**: 정시 TTS 알람 + 각 단계 개별 재생 버튼
- **모바일**: iOS autoplay 제한 우회(audio unlock 패턴) 적용

## 프로젝트 구조

```
Hi-Buddy.py                  # 메인 진입점 (홈 화면)
pages/
  1_코디네이터_일정입력.py    # 일정 설계 화면
  2_사용자_오늘_따라하기.py   # 따라하기 화면 (상시 표시)
utils/
  schedule_ai.py             # 자연어 → 일정 JSON 변환 (GPT)
  recipes.py                 # 초간단 레시피 DB + 운동 루틴
  image_ai.py                # 음식 이미지 검색 및 필터링
  youtube_ai.py              # 발달장애 친화 영상 검색
  tts.py                     # TTS 음성 합성
  runtime.py                 # 현재 활성 일정 계산
  config.py                  # API 키 관리
data/
  schedule_today.json        # 저장된 오늘 일정
  menu/, tools/              # 정적 이미지 에셋
```

## 실행 방법

```bash
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 환경변수 설정 (.env 또는 Streamlit secrets)
OPENAI_API_KEY=sk-...
GOOGLE_API_KEY=...
GOOGLE_CSE_ID=...
YOUTUBE_API_KEY=...

# 3. 실행
streamlit run Hi-Buddy.py
```

## 사용 흐름

```
1. 코디네이터가 "일정 만들기" 화면에서 오늘 일정 입력 후 저장
        ↓
2. 사용자가 "따라하기" 화면을 하루 종일 켜둠
        ↓
3. 정시마다 자동 알람 + 음성 안내
   → 요리 시간: 메뉴 선택 → 레시피 단계별 안내
   → 운동 시간: 루틴 선택 → 동작 단계별 안내
```

🍳 단계별 활동 안내
요리, 옷 입기, 운동 등 활동별로 단계를 나눠 보여주고 음성으로 읽어드려요.
존댓말 짧은 문장으로 한 번에 한 가지만 안내합니다.

💬 AI 도우미 대화
"저녁 뭐 먹지?", "오늘 일정이 뭐예요?" 같은 궁금한 것을 말하면 
냉장고 재료와 오늘 일정을 고려해서 대답해드려요.

📊 수행 기록 확인
이번 주에 얼마나 해냈는지 퍼센트가 아닌 "해낸 개수"로 따뜻하게 보여드립니다.

🚨 SOS 한 번의 터치
급할 때 SOS 버튼을 누르면 가족·복지관 선생님·119로 바로 연결돼요.

## 📊 6. 데이터 보안 설문 (Data Safety) 답변지

Play Console > 앱 콘텐츠 > 데이터 보안

### Q1. 앱이 사용자 데이터를 수집 또는 공유하나요?
**답변: 예 (Yes, but minimal)**

### Q2. 수집하는 데이터 유형
선택할 것 (아래 4개만):

**개인 정보 > 이름** ✅
- 수집 목적: 앱 내 맞춤 안내 (당사자 이름으로 부르기)
- 선택 사항: 선택 입력 가능
- 서버 전송: ❌ **기기 내 저장만**
- 암호화: 기기 저장 암호화

**개인 정보 > 전화번호** ✅ (긴급 연락처용)
- 수집 목적: 앱 내 기능 (SOS 연락처 저장)
- 선택 사항: 선택 입력 가능
- 서버 전송: ❌ **기기 내 저장만**
- 암호화: 기기 저장 암호화

**건강 및 피트니스 > 의료 정보** ✅ (약 알림)
- 수집 목적: 앱 내 기능 (약 복용 알림)
- 선택 사항: 선택 입력 가능
- 서버 전송: ❌ **기기 내 저장만**
- 암호화: 기기 저장 암호화

**앱 활동 > 앱 내 검색 기록** ✅ (AI 도우미 대화)
- 수집 목적: 앱 기능 제공 (AI 응답 생성)
- 선택 사항: 필수 (대화 시)
- 서버 전송: ✅ **백엔드 서버 일시 처리 (저장 안 함)**
- 암호화: HTTPS 전송 · 서버 무저장

### Q3. 데이터를 제3자에게 공유하나요?
**답변: 예 (LLM 제공업체)**
- Google (Gemini API) — AI 응답 생성 목적만
- 사용자 식별 정보 없이 전송

### Q4. 데이터 삭제 요청 처리
**답변: 사용자가 앱 내에서 직접 삭제 가능**
- 프로필 > 나의 정보 > 초기화 (향후 기능)
- 앱 삭제 시 모든 로컬 데이터 자동 삭제

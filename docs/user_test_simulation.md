## 핵심 인사이트

### 1. 장애 강도별 앱 활용 전략이 달라야 한다
- **경증**: 앱 혼자 사용 가능. 자립 도구로 기능.
- **중등도**: 초기 2~3주 코디네이터 동반 필요. 이후 부분 자립 가능.
- **중증**: 코디네이터 동반 필수. 앱은 "지시 도구"가 아니라 "함께하는 매개체" 역할.

### 2. TTS가 킬러 피처다
모든 장애 강도에서 TTS 음성 안내가 가장 긍정적 반응. 글자를 못 읽어도 음성은 이해.

### 3. B2B가 답이다
개인 보호자(B2C)보다 복지관/학교(B2B) 코디네이터들의 만족도가 더 높고, "기관용 라이선스" "다중 사용자 관리" 요청이 반복적으로 나옴.

### 4. 다음 버전 우선순위
1. Play Store 등록 (접근성 해소)
2. 다중 기기 동기화 (이미 백엔드 구현됨, 인증만 추가)
3. 일정 복사/반복 기능
4. 레시피 단계별 이미지
5. 기관용 대시보드

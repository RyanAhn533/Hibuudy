# 하루메이트 복지관 사용 매뉴얼

> 복지관 실무자를 위한 현장 사용 가이드 · v1.3.1 기준
> 이 문서 한 장으로 **10분 안에** 이용자 1명 세팅 가능

---

## 📌 한 눈에 보는 사용 흐름

```
① 담당자(선생님) 폰 설정       → 역할: "가족이나 학생을 위해"
                                   이름: 이용자 이름 (예: 유진)
                                   → 페어 코드 생성 (6자리)
      ↓
② 이용자 폰 설정                → 역할: "나 자신을 위해"
                                   이름: 본인 이름
                                   → ①의 페어 코드 입력
      ↓
③ 담당자: "일정 만들기"로 일과 입력 → AI가 자동 구조화
                                       → 저장
      ↓
④ 이용자: 앱 켜면 → 오늘 할 일 자동 표시 → 음성 안내
      ↓
⑤ 월말: 담당자 폰에서 "프로필 → 엑셀 내보내기" → 복지부 제출
```

---

## 1. 설치 (5분 · 1회)

### 담당자 폰

1. **Play Store** 열기
2. **"하루메이트"** 검색
3. **설치**
4. 앱 실행 → 아래 **"처음 설정" 따라하기**

### 이용자 폰 (해당 시)
- 담당자 폰과 같은 방식으로 설치
- 단, 역할 선택 시 **"나 자신을 위해"** 선택

---

## 2. 처음 설정 (3분)

### 담당자 입장

**Step 1. 역할 선택**
- "**가족이나 학생을 위해**" 선택
- "시작할게요" 탭

**Step 2. 이용자 정보**
- **이름**: 이용자 이름 입력 (예: `유진`)
- **어떤 상황인가요?**: `발달장애 · 자립생활` 선택 (기본)
- **사는 지역**: 드롭다운에서 선택 (예: 서울)
- **글자와 버튼 크기**: 이용자 수준에 맞게 `크게` 또는 `아주 크게`

**Step 3. 페어링 코드**
- 화면에 6자리 숫자가 나옴 (예: `379 416`)
- 이용자 폰 설정 시 이 코드를 사용
- **"연결됐어요"** 탭

---

## 3. 일정 만들기 (1일 2분)

### 매일 아침 또는 전날 저녁

1. 코디네이터 홈에서 **"일정 만들기"** 탭
2. 텍스트 편집창에 말하듯이 적기:
   ```
   08:00 · 기상하고 아침 먹기
   10:00 · 산책 30분
   12:00 · 점심 요리 (김치볶음밥)
   14:00 · 쉬는 시간
   16:00 · 운동 20분
   18:00 · 저녁 식사
   21:00 · 약 복용 · 잠들기
   ```
3. **"✨ 일정표 만들기"** 탭 (AI가 자동으로 단계·음성 안내 생성)
4. 결과 확인 후 **"저장하기"**
5. "저장 완료" 팝업 → **"오늘 하루 열기"** 탭
6. 이용자에게 폰을 건네거나, 공용 태블릿이라면 이용자가 사용 시작

---

## 4. 매일 사용 (이용자)

앱을 켜면:
- 지금 시간대 활동 **하나만** 크게 표시
- **음성으로 단계 안내** (스피커 버튼)
- 완료 시 **✓ 체크박스** 탭
- 다음 활동 시간 되면 **자동 전환 알림** (진동 + TTS)
- 어려울 땐 **SOS 버튼** (우하단) → 가족·복지관·119 즉시 연결

---

## 5. 엑셀 리포트 (월말 1회 · 1분)

### 복지부 제출용 자동 엑셀

1. 담당자 폰 → **프로필** (코디 홈 우하단)
2. 맨 아래 **📥 엑셀로 내보내기 (복지관 제출용)** 탭
3. 공유 시트 뜸 → **카카오톡 · 이메일 · 구글 드라이브** 중 선택
4. 파일 받으면 내용:
   - **시트 1 · 수행 기록**: 날짜별 활동·완료 여부·단계 진척
   - **시트 2 · 프로필**: 이용자 정보·약·연락처
   - **시트 3 · 이번 주 요약**: 활동별 완료율

## 6. 자주 묻는 질문

### Q1. 이용자가 앱을 잘못 눌러서 망가지면?
- SOS 버튼만 빨간색이라 주의 유도
- 나머지는 눌러도 저장 안 함 (단순 읽기)
- 최악의 경우 앱 껐다 켜면 원래대로

### Q2. 이용자 여러 명 담당할 때?
- **v2.0 (2026 Q3)**에 다중 이용자 관리 지원 예정
- 현재는 이용자 1명당 폰 1대

### Q3. 일정을 매일 새로 만들어야 하나요?
- 기본은 매일 저장
- **"내일도 같은 일정"** 버튼 탭 시 복사됨
- 요일별 고정 일정 기능은 v1.4 예정

### Q4. 인터넷 없어도 되나요?
- **대부분 기능 오프라인 작동** (저장된 일정·음성 안내·SOS)
- AI 일정 생성과 AI 도우미 대화만 인터넷 필요

### Q5. 이용자 개인정보 유출 걱정 없나요?
- **모든 데이터 사용자 기기에만 저장**
- AI 응답 생성 시 전송되는 대화는 서버에 저장 안 함
- 개인정보처리방침: https://ryanahn533.github.io/Hibuudy/privacy-policy.html

### Q6. 장애 강도별로 UI 바꿀 수 있나요?
- **일반 모드**: 일반 UI, 일반 글자
- **간단 모드**: 큰 버튼 2개만, 큰 글자
- **키오스크 모드**: 터치 불필요, 자동 진행 (중증용)
- 프로필 설정에서 변경 가능

### Q7. 음성 안내 끄기 가능?
- 각 단계별 재생 버튼 **누를 때만** 소리 재생
- 자동 재생은 "간단" "키오스크" 모드에서만
- 일반 모드는 자동 재생 X

3. 앱 설치 결정 심리
───────────────────────────────────────────────────────────────────

"설치할지 말지 10초 안에 결정한다."

설치 유도 포인트:
1. 아이콘이 **따뜻해 보이면** → 일단 설치
2. "발달장애"라는 단어가 노골적 → 설치 거부 ("남들이 볼까봐")
3. 앱 이름이 **평범하면** → 좋음 (하루메이트 ✓)
4. 설치 후 **3분 안에 뭐하는 앱인지** 파악 안 되면 → 삭제

"엄마는 기술 몰라. 아이콘 = 첫인상 = 전부."

───────────────────────────────────────────────────────────────────
4. 가치관
───────────────────────────────────────────────────────────────────

A. "우리 아이도 평범한 앱 쓰고 싶어 해"
   → 카카오톡·유튜브처럼 보이는 게 최고.
     "특수앱" 티 나면 준호가 창피해함.

B. "안심되어야 설치한다"
   → 파란색보다 **따뜻한 느낌**이 안심.
     병원 UI 같은 차가움 = 거부.

C. "엄마 없어도 혼자 쓸 수 있어야"
   → 복잡한 건 안 됨. 준호가 혼자 해야 함.

D. "정부 지원받는 앱 = 믿음"
   → "연세대 개발" "복지부 지원" 표시 있으면 신뢰 +50점.

E. "귀여우면 마음 놓인다. 근데 너무 애기 같으면 싫다."
   → 양면 감정. 균형이 핵심.

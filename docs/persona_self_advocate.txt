───────────────────────────────────────────────────────────────────
3. 가치관 (매우 중요)
───────────────────────────────────────────────────────────────────

A. "Nothing about us without us"
   → 당사자 없이 만든 앱 = 실패. 디자이너 혼자 상상한 UI 다 티남.

B. "평범하게 대우받고 싶다"
   → 카카오톡·인스타처럼 **일반 앱과 구별 안 되는** 것이 최고.
     "발달장애 전용" 브랜딩 = 모욕.

C. "보호자가 내 취향을 맘대로 짐작하지 마"
   → 엄마는 내가 귀여운 거 좋아한다고 생각 — 내가 결정해요.

D. "퍼즐 조각 절대 NO"
   → #RedInstead 운동 참고. 자폐를 "잃어버린 조각" 은유로 본 자선 단체
     의료 모델. 자기권익 운동에서 공식 반대.

E. "격려 오버 금지"
   → "와~ 잘했어요!!!" = 유치원생 취급 = 모욕.
     담백한 "잘했어요" 또는 "고생했어요"면 충분.

F. "통제보다 동반"
   → 보호자가 '지시하는' 앱은 싫음. '같이 하는' 앱이면 OK.
